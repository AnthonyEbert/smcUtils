smcUtils

utility functions for performing sequential Monte Carlo (SMC) otherwise known as particle filtering

Download .zip Download .tar.gz View on GitHub
library(devtools) # may need to install.packages("devtools") install_github("smcUtils","jarad")

This is an R package for sequential Monte Carlo (particle filtering) utilities. The package can be installed in R using install.packages("smcUtils") and loaded using library(smcUtils). The development version can be installed using install_github("smcUtils","jarad") where install_github is from the devtools package.
